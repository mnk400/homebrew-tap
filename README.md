# Mnk400 Tap

## How do I install these formulae?

`brew install mnk400/tap/<formula>`

Or `brew tap mnk400/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mnk400/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
